# Helport Assistant Demo

## 环境

Node: ^18.0.0 || ^20.0.0 || >=22.0.0

## 安装

```bash
npm install

# 如果你使用的是 yarn
yarn

# 如果你使用的是 pnpm
pnpm install
```

## 启动

```bash
npm run dev
```
